Match! Explore the World - A card matching game

A full stack web app that requires users to match country flags to the respective landmarks of the countries. This project was completed in 2 months for my 1st year module, Web Applications.

SkillsHTML, CSS, JS, Express.js, NeDB

Languages, Frameworks and Tools

  • Client Side: HTML, CSS, JavaScript
  • Server Side: Express.js, NeDB
  • Testing: Mocha Test
  • Best Practice & Accessibility: JSLint, Axe Tool

Details

The app is a card matching game that not only tests your memory but also improves your general knowledge. The overall theme is travelling, and users are expected to match a country landmark to their flag. The game should be relaxing so users can take their time to figure out which flag belongs to which country, hence "number of flips" was used as the score benchmark instead of "time taken to complete game".

The game scoreboard currently does not have an interface and is only displayed in raw JSON data. This could be improved in the future.
