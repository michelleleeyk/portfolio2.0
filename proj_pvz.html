Plant vs. Zombie, the Machine (Arduino)

A cardboard machine that provides users with a playful interaction. Contains various sensors and actuators, powered using an Arduino board and programmed in C. This project was completed in a month for my 2nd year module, Gizmo: Physical Computing.

SkillsArduino, Electronics, C

Project TypeUniversity - individual

View code on >GitHub

Project Brief

To create a playful cardboard machine that is inspired by a childhood favorite computer game, Plants vs. Zombies, thereby providing users with a nostalgic interaction. It should integrate machine elements (made of cardboard) with sensors and actuators that are connected to an Arduino board.

Due to Covid-19, students at Imperial were not able to use the machinery in the workshop, therefore cardboard was used as the main material of this project.

Process

  1. Ideation
    Devise general flow of game, identify all possible scenarios and list out all actions that the game should carry out.
  2. Concept development
    Dive deeper into each game action and convert them into detailed, actionable steps.
  3. Code & test independent functions
    Ensure that all game actions run correctly and smoothly.
  4. Low Fidelity Prototyping
    Build prototypes of mechanisms and other moving parts using cardboard.
  5. Combine and test code
  6. Build final cardboard model
  7. Integrate electronic components into cardboard model

Steps 3 & 4 were an iterative process.

Flow Chart

Key Actions

Zombie starts invading
The zombie is fixed onto a conveyor belt which is powered by a DC motor.

Pea shooter aims for and shoots zombie
The pea shooter is fixed onto a servo motor, which rotates back and forth along the vertical axis, between 0-90degrees. If the user presses the attack button when the pea shooter is aimed within 35-55degrees, the attack is successful.

Indicating that the pea shooter is shooting a pea
With each successful attack, a subsequent LED on the neopixel strip will light up, mimicking a pea moving through the air.

Ensuring the zombie returns to the original position
Once the reset button is pressed, the DC motor rotates in the opposite direction as before, while an ultrasonic sensor detects when the zombie is at the original position and stops the motor.
